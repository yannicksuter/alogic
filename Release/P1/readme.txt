Installation
------------

1. Da es sich um eine Java Applikation handelt, muss auf dem PC, auf dem das Programm gestartet werden soll, eine aktuelle Java JRE (Java Runtime Environment) ab Version 1.6 installiert sein. Auf http://www.java.com/de/download kann das neuste Installationspacket bezogen werden. 

Sollte eine JRE bereits installiert sein, kann dieser Punkt übersprungen werden.

2. Die Applikation selber bedingt keiner Installation. Es müssen lediglich alle Dateien aus dem Verzeichnis ..\Release\P1 in ein lokales Verzeichnis kopiert werden.

3. Die Applikation kann nun mittels start.cmd aufgerufen werden.

Tastenbelegung
--------------

In der 3D Sicht sind folgende Tasten belegt:

F1 : Export der berechneten Mantelung im OBJ Format in eine Datei
F2 : Export der berechneten Mantelung in die console
F9 : Randpunkte ein/ausblenden (Magenta/Punkte)
F10: Referenzobject ein/ausblenden (weiss schraffiert)
F11: Randlinien ein/ausblenden (gelb markiert)
